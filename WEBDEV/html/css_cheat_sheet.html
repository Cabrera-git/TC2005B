border

Modifies the box's entire border.


border-left, border-right, border-top, border-bottom

Modifies the specified part of the border, all sides can be independently manipulated.


border-radius

Smoothes the border's corners, be it by pixels, which curves the rectangle, or by percentage, which makes makes it more similar to a circle. Each corner can be manipulated independently, or it can be all done at once.


background-image

Adds a background to the corresponding div.


color

Changes the font color within the div, as well as changing the value of the currentcolor variable.


font

Changes the text's font and style.


font-family

Modifies the font family, i.e. Courier New, monospace, etc.


font-style, font-weight

Modifies font inclination (italic or oblique) and boldness.


width, height

Allows box resizing.


list-style

  • This works only
  • on lists.
  • Allows for list stylization
  • Such as index shape
  • and position.

margin

Adds a margin on all sides, it can also be used to set indiviudal margins per side.

padding

Similar to margin, but for use within an element.
